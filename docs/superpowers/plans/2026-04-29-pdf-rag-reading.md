# OZ AI — PDF/Document RAG Pipeline (Merged Final Plan)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Give OZ AI the ability to read, search, and answer questions from uploaded PDFs and DOCX files — both library Materials (lecture notes, research papers, course material) and Books with a `pdfUrl`. Materials are chunked and indexed into PostgreSQL full-text search. Books store extracted text as a searchable blob. All extraction happens asynchronously so uploads are never blocked.

**Architecture:**
- **Materials** → `MaterialIndexerService` extracts text on approval, splits into 400-word overlapping chunks stored in a `MaterialChunk` table. `MaterialSearchService` runs `tsvector`/`plainto_tsquery` ranked search over those chunks. OZ gets `search_study_material` and `list_study_materials` tools.
- **Books** → Same `MaterialIndexerService` has an `indexBook()` method that fetches the `pdfUrl`, extracts text, and stores it as an `extractedText` blob on the `Book` row. `read_ebook` checks this blob before falling back to an HTTP fetch.
- All statuses tracked with a shared `IndexStatus` enum. Frontend shows index state in admin and public pages.

**Tech Stack:** `pdf-parse@2.4.5` (installed), `mammoth@1.12.0` (installed), NestJS 10, Prisma 5, PostgreSQL (`tsvector`/`plainto_tsquery` — no external vector DB), Next.js 14 Tailwind.

- [ ] **Step 1: Create the file**

```typescript
// apps/api/src/materials/material-indexer.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { PrismaService } from '../prisma/prisma.service';
import { IndexStatus } from '@prisma/client';
import * as fs from 'fs/promises';
import * as path from 'path';
import * as mammoth from 'mammoth';

// pdf-parse is CommonJS — require() avoids missing-types error
// eslint-disable-next-line @typescript-eslint/no-var-requires
const pdfParse = require('pdf-parse') as (
  buffer: Buffer,
) => Promise<{ text: string }>;

const CHUNK_WORDS   = 400;  // target chunk size in words
const OVERLAP_WORDS = 40;   // overlap carried into next chunk
const MIN_CHUNK_CHARS = 30; // discard noise chunks shorter than this

@Injectable()
export class MaterialIndexerService {
  private readonly logger = new Logger(MaterialIndexerService.name);

  constructor(private readonly prisma: PrismaService) {}

  // ── Materials ────────────────────────────────────────────────────

  /**
   * Extract text from a material's file, split into chunks, store as
   * MaterialChunk rows. Call fire-and-forget — never await from HTTP handlers.
   */
  async indexMaterial(materialId: string): Promise<void> {
    const material = await this.prisma.material.findUnique({
      where: { id: materialId },
    });
    if (!material) {
      this.logger.warn(`indexMaterial: material ${materialId} not found`);
      return;
    }
    if (!material.fileUrl) {
      await this.setMaterialStatus(materialId, IndexStatus.NOT_APPLICABLE);
      return;
    }
    const ext = path.extname(material.fileUrl).toLowerCase();
    if (!['.pdf', '.docx', '.doc', '.txt'].includes(ext)) {
      await this.setMaterialStatus(materialId, IndexStatus.NOT_APPLICABLE);
      return;
    }

    await this.setMaterialStatus(materialId, IndexStatus.PROCESSING);
    try {
      const buffer = await this.fetchFile(material.fileUrl);
      const rawText = ext === '.pdf'
        ? await this.extractPdf(buffer)
        : ext === '.txt'
          ? buffer.toString('utf-8')
          : await this.extractDocx(buffer);

      const chunks = this.chunkText(rawText);
      if (chunks.length === 0) {
        this.logger.warn(`Material ${materialId}: 0 chunks after extraction`);
        await this.setMaterialStatus(materialId, IndexStatus.FAILED);
        return;
      }

      await this.prisma.materialChunk.deleteMany({ where: { materialId } });
      await this.prisma.materialChunk.createMany({
        data: chunks.map((content, chunkIndex) => ({
          materialId,
          chunkIndex,
          content,
          tokenCount: Math.ceil(content.length / 4),
        })),
      });
      await this.setMaterialStatus(materialId, IndexStatus.INDEXED);
      this.logger.log(
        `✅ Indexed material ${materialId} "${material.title}": ${chunks.length} chunks`,
      );
    } catch (err) {
      this.logger.error(`❌ Failed to index material ${materialId}: ${String(err)}`);
      await this.setMaterialStatus(materialId, IndexStatus.FAILED);
    }
  }

  // ── Books ────────────────────────────────────────────────────────

  /**
   * Fetch a book's pdfUrl, extract full text, store on the Book row.
   * Call fire-and-forget — never await from HTTP handlers.
   */
  async indexBook(bookId: string): Promise<void> {
    const book = await this.prisma.book.findUnique({ where: { id: bookId } });
    if (!book) {
      this.logger.warn(`indexBook: book ${bookId} not found`);
      return;
    }
    if (!book.pdfUrl) {
      await this.setBookStatus(bookId, IndexStatus.NOT_APPLICABLE);
      return;
    }

    await this.setBookStatus(bookId, IndexStatus.PROCESSING);
    try {
      const buffer = await this.fetchFile(book.pdfUrl);
      const rawText = await this.extractPdf(buffer);
      const cleaned = this.cleanText(rawText);

      await this.prisma.book.update({
        where: { id: bookId },
        data: {
          extractedText: cleaned || null,
          indexStatus: cleaned ? IndexStatus.INDEXED : IndexStatus.FAILED,
        },
      });
      this.logger.log(
        `✅ Indexed book ${bookId} "${book.title}": ${cleaned.length} chars`,
      );
    } catch (err) {
      this.logger.error(`❌ Failed to index book ${bookId}: ${String(err)}`);
      await this.setBookStatus(bookId, IndexStatus.FAILED);
    }
  }

  // ── Private helpers ─────────────────────────────────────────────

  private async setMaterialStatus(materialId: string, status: IndexStatus): Promise<void> {
    await this.prisma.material.update({ where: { id: materialId }, data: { indexStatus: status } });
  }

  private async setBookStatus(bookId: string, status: IndexStatus): Promise<void> {
    await this.prisma.book.update({ where: { id: bookId }, data: { indexStatus: status } });
  }

  /**
   * Fetch file as Buffer from an S3 URL (https) or a local Multer path (/uploads/…).
   */
  private async fetchFile(fileUrl: string): Promise<Buffer> {
    if (fileUrl.startsWith('http://') || fileUrl.startsWith('https://')) {
      const res = await fetch(fileUrl, {
        signal: AbortSignal.timeout(30_000),
        headers: { 'User-Agent': 'LibrarySystem-Indexer/1.0' },
      });
      if (!res.ok) throw new Error(`HTTP ${res.status} fetching ${fileUrl}`);
      return Buffer.from(await res.arrayBuffer());
    }
    // Local disk: fileUrl is like /uploads/materials/uuid.pdf
    return fs.readFile(path.join(process.cwd(), fileUrl));
  }

  private async extractPdf(buffer: Buffer): Promise<string> {
    const data = await pdfParse(buffer);
    return this.cleanText(data.text ?? '');
  }

  private async extractDocx(buffer: Buffer): Promise<string> {
    const result = await mammoth.extractRawText({ buffer });
    return this.cleanText(result.value ?? '');
  }

  private cleanText(raw: string): string {
    return raw
      .replace(/\r\n/g, '\n')
      .replace(/[ \t]+/g, ' ')
      .replace(/\n{3,}/g, '\n\n')
      .trim();
  }

  /**
   * Split plain text into overlapping word-window chunks.
   */
  private chunkText(text: string): string[] {
    const words = text
      .replace(/\n+/g, ' ')
      .split(/\s+/)
      .filter((w) => w.length > 0);

    const chunks: string[] = [];
    let start = 0;

    while (start < words.length) {
      const end = Math.min(start + CHUNK_WORDS, words.length);
      const chunk = words.slice(start, end).join(' ').trim();
      if (chunk.length >= MIN_CHUNK_CHARS) chunks.push(chunk);
      if (end === words.length) break;
      start += CHUNK_WORDS - OVERLAP_WORDS;
    }

    return chunks;
  }

  /**
   * Return a relevance-matched excerpt from a book's full extracted text.
   * Used by the read_ebook upgrade in agent.service.ts.
   */
  getRelevantExcerpt(fullText: string, question: string, maxChars = 4_500): string {
    if (fullText.length <= maxChars) return fullText;

    const keywords = question
      .toLowerCase()
      .split(/\s+/)
      .filter((w) => w.length > 3);

    for (const kw of keywords) {
      const idx = fullText.toLowerCase().indexOf(kw);
      if (idx !== -1) {
        const start = Math.max(0, idx - 300);
        const end = Math.min(fullText.length, start + maxChars);
        return (
          fullText.substring(start, end) +
          '\n\n[Excerpt — matched section. Full document in library.]'
        );
      }
    }

    return (
      fullText.substring(0, maxChars) +
      '\n\n[Excerpt — beginning of document. Ask about a specific section for more.]'
    );
  }
}
```

// apps/api/src/materials/material-search.service.ts
import { Injectable } from '@nestjs/common';
import { PrismaService } from '../prisma/prisma.service';
import { IndexStatus } from '@prisma/client';

export interface ChunkSearchResult {
  id: string;
  materialId: string;
  chunkIndex: number;
  content: string;
  tokenCount: number;
  material: {
    title: string;
    type: string;
    authorName: string;
    facultyCode: string | null;
    courseCode: string | null;
  };
}

export interface IndexedMaterialSummary {
  id: string;
  title: string;
  type: string;
  authorName: string;
  facultyCode: string | null;
  courseCode: string | null;
  chunkCount: number;
}

@Injectable()
export class MaterialSearchService {
  constructor(private readonly prisma: PrismaService) {}

  /**
   * Ranked full-text search across all indexed, published, approved material chunks.
   * Uses PostgreSQL tsvector/tsquery — no external vector DB required.
   */
  async searchChunks(
    query: string,
    options: { materialId?: string; facultyCode?: string; limit?: number } = {},
  ): Promise<ChunkSearchResult[]> {
    const safeLimit = Math.min(options.limit ?? 5, 10);

    const conditions: string[] = [
      `m."isApproved" = true`,
      `m."isPublished" = true`,
      `m."indexStatus" = 'INDEXED'`,
      `to_tsvector('english', mc.content) @@ plainto_tsquery('english', $1)`,
    ];
    const params: unknown[] = [query];
    let paramIdx = 2;

    if (options.materialId) {
      conditions.push(`mc."materialId" = $${paramIdx++}`);
      params.push(options.materialId);
    }
    if (options.facultyCode) {
      conditions.push(`m."facultyCode" = $${paramIdx++}`);
      params.push(options.facultyCode);
    }

    const sql = `
      SELECT
        mc.id,
        mc."materialId",
        mc."chunkIndex",
        mc.content,
        mc."tokenCount",
        m.title         AS "materialTitle",
        m.type          AS "materialType",
        m."authorName",
        m."facultyCode",
        m."courseCode"
      FROM material_chunks mc
      JOIN materials m ON m.id = mc."materialId"
      WHERE ${conditions.join(' AND ')}
      ORDER BY
        ts_rank(to_tsvector('english', mc.content), plainto_tsquery('english', $1)) DESC
      LIMIT ${safeLimit}
    `;

    const rows = await this.prisma.$queryRawUnsafe<Array<{
      id: string; materialId: string; chunkIndex: number; content: string;
      tokenCount: number; materialTitle: string; materialType: string;
      authorName: string; facultyCode: string | null; courseCode: string | null;
    }>>(sql, ...params);

    return rows.map((row) => ({
      id: row.id,
      materialId: row.materialId,
      chunkIndex: row.chunkIndex,
      content: row.content,
      tokenCount: row.tokenCount,
      material: {
        title: row.materialTitle,
        type: row.materialType,
        authorName: row.authorName,
        facultyCode: row.facultyCode,
        courseCode: row.courseCode,
      },
    }));
  }

  /**
   * List all materials that are indexed and visible — used by list_study_materials tool.
   */
  async listIndexedMaterials(
    options: { facultyCode?: string; limit?: number } = {},
  ): Promise<IndexedMaterialSummary[]> {
    const materials = await this.prisma.material.findMany({
      where: {
        isApproved: true,
        isPublished: true,
        indexStatus: IndexStatus.INDEXED,
        ...(options.facultyCode ? { facultyCode: options.facultyCode } : {}),
      },
      select: {
        id: true, title: true, type: true, authorName: true,
        facultyCode: true, courseCode: true,
        _count: { select: { chunks: true } },
      },
      orderBy: { createdAt: 'desc' },
      take: Math.min(options.limit ?? 20, 50),
    });

    return materials.map((m) => ({
      id: m.id, title: m.title, type: m.type,
      authorName: m.authorName, facultyCode: m.facultyCode, courseCode: m.courseCode,
      chunkCount: m._count.chunks,
    }));
  }
}

- **Vector embeddings / semantic search** — `tsvector` is keyword-based. Semantic similarity (pgvector or Pinecone) is a follow-up once the pipeline is stable.

- **Streaming progress to frontend** — `PROCESSING` status is polled, not pushed. WebSocket/SSE would enable real-time progress bars.
